# Projeto: Pesquisa de Preços

Este diretório contém os códigos-fonte que automatizam a pesquisa, o scraping e a geração de planilhas de preços a partir de sites listados.

## Arquivos Principais
*   `main.py`, `pesquisa_preco.py`, `pesquisa_preco 2.py`: Scripts base em Python construídos usando as bibliotecas pertinentes para raspagem de dados e análise (ex: Beautiful Soup, Selenium, pandas).
*   `reqs.txt`, `reqs_utf8.txt`: Arquivos listando os requisitos instalados no ambiente virtual do projeto.
*   `extracao_precos_...csv`, `produtos.csv`: Resultados gerados ou matrizes base utilizados pelos scripts.
*   `.env.example`, `env`: Exemplos e configurações das chaves / environment variables necessárias para execução.
